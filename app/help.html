yt2rss — YouTube to RSS

Purpose: Convert YouTube channels or videos into RSS feeds suitable for podcast apps.

Subscribe to YouTube channels as a podcast, download videos on the fly, no storage required.

API Endpoints

  • /{{CHANNEL_URL}}.rss — Returns an RSS feed for the channel.
  • /video/{{VIDEO_ID}}.mp4 — Streams the video directly for podcast apps.
  • Firefox extension
  • Chrome-compatible extension

Supports channel URLs like @handle (served via /_user/handle.rss) or /c/Name.

Documentation

Interactive API docs available at /docs (Swagger UI).

Usage

  1. Install subcRiSS Firefox extension (or Chrome version).
    • Open Firefox and go to about:debugging#/runtime/this-firefox
    • Click "Load Temporary Add-on".
    • Select subcRiSS.zip or manifest.json.
  2. Configure your NAS base URL in the extension options.
  3. Visit a YouTube video or channel, click the button, and get your RSS feed URL.
